Cara Endorse

Ikuti langkah-langkah mudah berikut untuk mempromosikan server Minecraft Anda

1

Pilih Paket

Pilih paket endorse yang sesuai dengan kebutuhan dan budget Anda. Kami menyediakan berbagai pilihan:

  • TikTok - Untuk konten short video viral
  • YouTube - Untuk review detail server
  • WhatsApp Channel - Untuk komunitas
  • Bundling - Hemat dengan paket kombinasi
Lihat Price List
2

Lihat Jadwal

Cek ketersediaan jadwal upload di halaman jadwal kami. Pilih tanggal yang masih tersedia (Available).

  • Jadwal ditampilkan 20 hari ke depan
  • Jam upload default: 19:00 WIB
  • Slot terbatas, cepat habis!
Cek Jadwal
3

Ambil Slot

Klik tombol "Ambil Slot" pada tanggal yang Anda inginkan. Anda akan diarahkan ke WhatsApp dengan pesan otomatis.

Tips: Segera konfirmasi setelah mengirim pesan untuk mengamankan slot Anda.

4

Hubungi via WhatsApp

Setelah diarahkan ke WhatsApp, tim kami akan membalas dan memandu Anda melalui proses selanjutnya.

  • Konfirmasi ketersediaan slot
  • Diskusi brief konten
  • Penjelasan detail paket
Chat WhatsApp
5

Isi Data Server

Siapkan informasi lengkap tentang server Minecraft Anda untuk kami review:

  • Nama Server - Nama unik server Anda
  • IP / Link - IP server atau link invite
  • Deskripsi - Fitur unik dan selling point
  • Versi Minecraft - 1.8, 1.16, 1.20, dll
  • Gambar/Logo - (opsional) untuk WhatsApp
6

Pembayaran

Lakukan pembayaran di awal untuk mengamankan slot Anda. Kami menerima berbagai metode pembayaran:

  • Transfer Bank (BCA, Mandiri, BNI)
  • E-Wallet (DANA, OVO, GoPay, ShopeePay)
  • QRIS

Catatan: Pembayaran 100% di awal. Refund hanya jika kesalahan dari pihak kami.

7

Recording H-1

Recording gameplay akan dilakukan H-1 (satu hari sebelum jadwal upload). Pastikan server Anda dalam kondisi optimal.

  • Server online dan stabil
  • Fitur yang dijanjikan berfungsi
  • Staff siap membantu jika diperlukan

Info: Waktu recording fleksibel, akan dikonfirmasi sebelumnya.

8

Upload Sesuai Jadwal

Video akan diupload sesuai jadwal yang telah ditentukan. Anda akan menerima notifikasi saat konten sudah live.

  • Notifikasi saat video upload
  • Link konten akan dikirim
  • Monitor engagement dan feedback

Selamat! Server Anda sekarang mendapatkan exposure lebih banyak!

Siap Memulai?

Langsung cek jadwal dan amankan slot untuk server Anda sekarang juga!
